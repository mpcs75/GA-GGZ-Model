Generieke Architectuur GGZ
Dit model heeft als basis een op maat gemaakte weergave van het Referentie Domeinmodel voor de GGZ versie 1. Deze basis heb ik uitgebreid en verder op maat gemaakt. Wat ik daarbij geprobeerd heb is om niet enkel een aantal logisch gegroepeerde set aan handelingen en informatie verwerking weer te geven maar alles weer te geven vanuit de context van dienstverlening, inclusief het managen daarvan. - Dit raamwerk geeft een architectuur landschap weer zoals dat er uit zou kunnen zien binnen een GGZ instelling. - Dit raamwerk is bedoeld als vrijblijvend hulpmiddel voor beoefenaars van digitale architectuur binnen een GGZ instelling. Bronnen: - RDG: https://nictiz.nl/standaarden/referentiedomeinenmodellen/referentiedomeinmodel-ggz/ - USM: Git repo voor comments : https://github.com/mpcs75/GA-GGZ-Model ((toegang is nodig)
