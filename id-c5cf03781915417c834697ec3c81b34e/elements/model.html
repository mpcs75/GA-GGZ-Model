Generieke Architectuur GGZ
Dit model heeft als basis een op maat gemaakte weergave van het Referentie Domeinmodel voor de GGZ versie 1. In dit model worden informatiedomeinen, bedrijfsarctiviteiten, en verwerkte informatie beschreven. Deze drie onderdelen is aangevuld met 14 extra onderdelen om tot een compleet generiek raamwerk te komen. Dit raamwerk geeft een archictuur landschap weer zoals dat er uit zou kunnen zien binnen een GGZ instelling. Dit raamwerk is bedoeld als vrijblijvend hulpmiddel voor beoefenaars van digitale architectuur binnen een GGZ instelling. Bron RDG onderwerpen: https://nictiz.nl/standaarden/referentiedomeinenmodellen/referentiedomeinmodel-ggz/ Git repo voor comments (toegang is nodig): https://github.com/mpcs75/GA-GGZ-Model
